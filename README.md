## 前言

deepseek 网页版和手机 app 的历史会话都是堆在一个列表里（模型版本：DeepSeek-V3.2），也没有像 chatgpt 一样的分项目存放，要快速查询历史会话内容只能导出。问题在于直接导出的是一个全部会话整合的人类不易读的 JSON 文件，需要转换一下格式。

搜索了下市面上有浏览器插件可以一键导出文档，例如 Deepseek Chat Exporter ，但是一次只能导出一份会话，并且我看了下我导出为 md 文档时，使用 Typora 阅读时代码块格式直接崩坏。还有一款工具：https://github.com/xionglongztz/DeepSeekDataExporter  可以导出无误，但缺少了些细节处理，索性自己加些需求也让 AI 写了个脚本转换成 md 文档。



## 环境需求

- 有 安装 python 环境。
- 从 deepseek 导出了会话文件 `conversations.json`。



## 脚本功能

- 把每个会话（含多个对话）分别生成为单个 md 文件，文件名与 app 里的会话标题一致。
- 把会话里所有的一级标题删除。
- 全文从 `## 👤 用户` 开始。
- 若有深度思考内容，深度思考内容像 app 里一样放到引用里，位于 ` ## 🤖 DeepSeek（模型思考）` 下面。
- 正文内容在 `## 🤖 DeepSeek（最终回答）` 下面。
- 多个对话重复这样的标题和内容编排，查阅上方便定位。
- 额外生成一个 `index.md` 文件，文件内容按历史会话从旧到新排序，可点击跳转到对应文件。按需浏览。



## 文档预览

```python
## 👤 用户
瞎 bb

## 🤖 DeepSeek（模型思考）
> ……

## 🤖 DeepSeek（最终回答）
正文剩余内容（不再包含一级标题）
```



## 使用方法

1. 在导出的文件的目录下打开终端；

2. 运行：

   ```python
   py json_to_md.py
   ```

3. 运行结束后当前目录将出现 `deepseek_conversations` 文件夹，里面即存放所有历史会话和 `index.md`。
